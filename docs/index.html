About Me

I'm the lead developer of the Meadowlark application, which aims to be a fully-featured, free and open source DAW (Digital Audio Workstation). I'm also the lead developer of the RustyDAW project, which is what Meadowlark is built on.

If you want to chat, I am most active on the Meadowlark Discord Server and the Rust Audio Discord Server. There are also a lot of other cool Rusty developers there as well.
